# 客户端配置

服务端地址：

```txt
https://now.m1301.cyou
```

## Windows

对应文件：

- `windows-pc-1.config.json`
- `windows-pc-2.config.json`

用法：

1. 下载 `live-dashboard-agent.exe`
2. 把对应设备的配置文件复制到 `.exe` 同目录
3. 重命名为 `config.json`
4. 双击运行

## Android

对应文件：

- `android-1.settings.txt`
- `android-2.settings.txt`

用法：

1. 安装 Android App
2. 打开 `SetupScreen`
3. 按对应文件填写 `server_url`、`token`、`report_interval`
4. 保存后点击“开始监听”
